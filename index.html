Diferentes tipos de Seletor

1. *

Todos os elementos em uma página


* {
  margin: 0;
  padding: 0;
}

Ou todos elementos filho de um pai


#container * {
 border: 1px solid black;
}

Exemplo

2. #X

Seletor por ID


#container {
 width: 960px;
 margin: auto;
}

Exemplo

3. .X

Seletor de classe


.error {
 color: red;
}

Exemplo

4. X

Seletor por tipo "Tag"


a { color: red; }
ul { margin-left: 0; }

Exemplo

5. X Y

Seletor de descendência


li a {
 text-decoration: none;
}

.header .nav {
 background: #CCC;
}

Exemplo

Diferentes tipos
de Pseudo Classes

1. X:hover

Ação de hover no elemento


div:hover {
 background: #e3e3e3;
}

a:hover {
 background: #e3e3e3;
}

Exemplo

2. X:focus

Ação de focus no elemento


div:focus {
 background: #e3e3e3;
}

a:focus {
 background: #e3e3e3;
}

Exemplo

3. X:visited

Link que já foi acessado


a:visited { color: #4b2f89; }
a:visited { background-color: white }

4. X:active

Irá atuar na estilização DURANTE o ato de clicar


a:active { 
 color: #FFF; 
}

5. X:before Ou X:after

É frequentemente utilizado para adicionar conteúdo antes ou depois


p:before { 
  content: "«";
  color: blue;
}

.template:after { 
  content: "»";
  color: red;
}

Exemplo

6. X:nth-child(n)

Seleciona o filho na posição de (N)


li:nth-child(3) {
 color: red;
}

Exemplo

7. X:nth-of-type(n)

Seleciona o elemento na posição de (N)


ul:nth-of-type(3) {
 border: 1px solid black;
}

Exemplo

8. X:first-child

Permite-nos selecionar o primeiro elemento do conjunto alvo


ul li:first-child {
 border-top: none;
}

9. X:last-child

Permite-nos selecionar o último elemento do conjunto alvo


ul li:last-child {
 color: #CCC;
}

3. display

block

Ocupa 100% da largura do elemento pai


.element li {
  display: block;
}

inline

Ocupa a largura total do seu conteúdo apenas


.element li {
  display: inline;
}

inline-block

Desloca seus elementos para uma posição horizontal, colocando-os um ao lado do outro

*Aceita margin-top, margin-bottom, width e height


.element li {
  display: inline-block;
}
